開始使用（第一次設定）

1

開啟系統網址

建議使用 Chrome 瀏覽器，輸入管理員提供的網址

2

貼上 API 網址

首次開啟會出現設定畫面，貼上管理員提供的 API 網址後點「連線並開始使用」

3

確認連線成功

右上角出現綠點「已同步」代表連線成功 ✅。之後打開不需要再設定

💡 若右上角出現連線失敗，點「重新整理」按鈕，或點左側底部「⚙ 修改 API 設定」重新貼上網址。

週會任務管理

填入個人待辦事項

  1. 1找到自己的名字卡片
  2. 2在輸入框輸入待辦事項，按 Enter 或點「新增」
  3. 3自動儲存並同步給所有人

任務操作

操作方式
勾選完成點任務左側勾選框，再點一次取消
修改內容直接點任務文字編輯，按 Enter 或點其他地方儲存
刪除任務滑鼠移到任務上，點右側出現的 ✕
拖移排序滑鼠移到任務上，拖曳左側六點圖示
貼入網址直接貼網址，自動轉為可點擊連結
附加連結點輸入框下方「🔗 附加超連結」，分別填名稱和網址

週次管理

操作方式
切換週次點左側欄位的週次名稱
新增週次點左側「＋ 新增本週」，輸入名稱後建立
修改名稱點上方工具列的週次名稱直接編輯，Enter 儲存
刪除週次滑鼠移到左側週次上，點右側 ×，確認後刪除（不可復原）

成員管理

操作方式
新增成員點右上角「＋ 新增成員」，新增後同步到所有六個模組
編輯成員點成員頭像，可修改名稱、顏色、上傳大頭貼
刪除成員點頭像進入編輯，點「刪除成員」（該成員任務一併刪除）

週會討論事項（畫面左側欄位）

操作方式
輸入文字直接點擊欄位輸入，離開後自動儲存
插入連結點右上角「插入連結」，填入文字和網址
插入圖片點「插入圖片」選擇檔案，或直接 Ctrl+V 貼截圖，或拖曳圖片進欄位
放大圖片點擊圖片放大，點空白處或右上角 × 關閉
移除圖片滑鼠移到圖片，點右上角 ×
💡 圖片上傳後所有人都看得到（透過圖床同步）。

整合總覽

點右上角「總覽」切換，查看所有成員本週任務和完成進度統計。

員工請假

申請請假

  1. 1點右上角「申請請假」按鈕
  2. 2選擇姓名、假別，填入起始和結束日期
  3. 3天數自動計算（只算週一到週五），可手動調整（半天填 0.5）
  4. 4填寫事由（選填），點「提交申請」

可選假別（共 11 種）

事假普通傷病假婚假 喪假產假產檢假 陪產檢及陪產假生理假 家庭照顧假公假特別休假

編輯 / 刪除

點任一筆紀錄右側的 ✏️ 編輯，或 🗑️ 刪除（會跳出確認視窗）。

篩選

上方篩選列可依成員篩選，查看特定人的請假紀錄。

🕐

班表 / 出勤行事曆

月曆

  • 上方左右箭頭切換月份
  • 請假、排休、公出紀錄顯示在對應日期格子上
  • 點擊任一天格子，彈出當日所有出勤狀況（請假 / 排休 / 公出）
  • 週六日底色為淺灰色

排休 / 公出

功能操作方式
新增排休點月曆下方「排休」頁籤 → 「＋ 新增排休」，填入姓名、日期、備注
新增公出點月曆下方「公出」頁籤 → 「＋ 新增公出」，填入姓名、日期、備注
編輯 / 刪除點各筆紀錄右側的 ✏️ 或 🗑️
💡 排休和公出只顯示當月份的紀錄，切換月份後自動更新。

費用報支

提交費用申請

  1. 1選擇申請人、填入申請日期
  2. 2選擇費用類型（差旅費 / 膳食費 / 交際應酬費 / 辦公雜支）
  3. 3填入費用摘要、預計/實際發生期間
  4. 4差旅費需額外填寫出差地點
  5. 5填寫報銷明細（名稱、類別、金額），可新增多筆，不限數量
  6. 6上傳憑證照片（發票、收據、登機證等）
  7. 7勾選是否已附上所有憑證，點「提交申請」
📧 提交後系統自動寄出通知信給管理員。

合格報支項目說明

類型說明
差旅費機票、高鐵、台鐵、計程車、大眾運輸、出差住宿（標準單/雙人房）
膳食費招待客戶之餐敘，或符合規定之差旅每日膳雜費
交際費維繫業務關係或專案推動所必需之應酬開銷
辦公雜支經主管事前核准之耗材、書籍、郵電費等

查看歷史紀錄

右側顯示所有申請紀錄，可依成員、費用類型、狀態、日期區間篩選。點任一筆展開查看完整明細和憑證照片，管理員可更改狀態（待審核 / 已核准 / 已退回）。

員工通訊錄

查看同事資訊

以卡片形式顯示所有成員的聯絡資訊，可在上方搜尋框輸入姓名、職稱或部門快速查找。點任一卡片查看完整資訊，可直接點電話撥打或點 Email 寄信。

編輯自己的資訊

  1. 1點右上角「✏ 編輯我的資訊」
  2. 2選擇自己的名字
  3. 3填入職稱、部門、電話、Email、LINE ID、生日（例：03/15）
  4. 4點「儲存」，資訊同步給所有人

新增成員

點「＋ 新增成員」，輸入姓名和選擇顏色後建立。新成員會同步出現在所有六個模組（任務、請假、班表、報支、通訊錄、福利）。

常見問題

右上角顯示「連線失敗」怎麼辦？
確認網路連線正常後點「重新整理」。若持續失敗，點左側欄位最底部「⚙ 修改 API 設定」，重新貼上管理員提供的 API 網址即可。
我填的資料其他人看不到？
右上角顯示「已儲存」才代表同步成功。若一直轉圈或顯示「儲存失敗」，稍等幾秒後系統會自動重試，或手動按「重新整理」再試一次。
新增成員後其他模組沒有出現？
切換到其他模組再切回來即可，或點右上角「重新整理」按鈕重新載入資料。
可以在手機上使用嗎？
可以，用手機瀏覽器開啟網址即可使用。點左上角選單圖示可開啟模組導覽列，版面會自動調整為單欄適合手機閱讀。
資料會不見嗎？
不會。所有資料存在公司 Google 帳號的 Sheets 試算表裡，關閉瀏覽器或重新整理頁面都不影響資料。
忘記 API 網址怎麼辦？
點左側欄位最底部「⚙ 修改 API 設定」，貼上管理員提供的 API 網址即可重新連線。API 網址格式為：https://script.google.com/macros/s/...../exec
圖片上傳失敗怎麼辦？
系統會自動嘗試多個圖床，若全部失敗會暫存在你的瀏覽器本機（其他人看不到）。請確認網路正常後重試，或聯繫管理員。
